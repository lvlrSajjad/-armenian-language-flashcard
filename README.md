### Before Deploying

Update the following files with your actual domain:
1. `index.html` - Replace `https://armenian-language-flashcard.web.app/` with your domain
2. `public/sitemap.xml` - Update domain and lastmod date
3. `public/robots.txt` - Update sitemap URL
4. Add social media images: `og-image.jpg`, `twitter-image.jpg`, `icon-192x192.png`, `icon-512x512.png`
